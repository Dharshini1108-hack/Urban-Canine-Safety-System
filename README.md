# Urban Canine Safety Belt and Monitoring System

## Project Overview
A smart IoT-based safety belt for street dogs that monitors health and reduces dog bite incidents.

## Problem Statement
Increasing dog bite cases and delayed disease detection in street dogs.

## Solution
The belt monitors temperature and humidity, tracks GPS location, and uses ultrasonic and pheromone-based deterrents.

## Features
- GPS Tracking
- Temperature Monitoring
- Humidity Monitoring
- Ultrasonic Deterrent
- Bluetooth Monitoring

## Technology Stack
- Arduino Nano / ESP32
- C++
- Bluetooth
- TinyGPS++
- DHT11 Sensor
