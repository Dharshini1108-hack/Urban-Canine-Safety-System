# Urban Canine Safety Belt and Monitoring System

## Project Overview
The Urban Canine Safety Belt is an IoT-based wearable device designed to reduce dog bite incidents in urban areas through a safe and humane approach. The system detects potentially aggressive situations using predefined physical triggers and automatically activates an ultrasonic sound to discourage aggressive behavior without harming the animal. By providing real-time intervention, the device promotes safer human-animal interaction and supports ethical management of street dogs.

## Problem Statement
Street dog attacks are a major public safety concern due to increasing dog bite incidents and the risk of rabies transmission. Lack of monitoring, delayed vaccination, and frequent human–street dog interactions contribute to this issue. Existing solutions mainly focus on treatment after incidents rather than prevention. This project aims to provide a preventive approach using a humane ultrasonic mechanism to reduce aggressive dog behavior while ensuring animal welfare.

## Proposed Solution
The Urban Canine Safety Belt is an IoT-based wearable device attached to dogs for monitoring their behavior and movement. The system uses motion and ultrasonic sensors to identify aggressive patterns and activates a non-lethal ultrasonic sound to distract the dog. If extreme aggression is detected, a pheromone spray mechanism is triggered as an additional calming deterrent. The device also includes GPS tracking, health monitoring, and Bluetooth connectivity for real-time monitoring through a mobile application.

## Features

- **Behavior Detection:** Motion and ultrasonic sensors detect potentially aggressive dog behavior in real time.
- **Ultrasonic Deterrent:** Emits a safe, non-lethal ultrasonic sound to discourage aggressive behavior.
- **Pheromone Spray:** Releases pheromone spray when extreme aggression continues.
- **GPS Tracking:** Monitors dog movement and identifies high-risk roaming areas.
- **Health Monitoring:** Detects unusual behavior patterns and possible health issues.
- **Bluetooth Integration:** Enables real-time monitoring and alerts through a mobile application.
- **Portable Wearable Design:** Lightweight and suitable for daily use.

## Technology Stack

- **Hardware:** ESP32, Ultrasonic Sensor, PIR Motion Sensor, GPS Module, Bluetooth (BLE), Pheromone Spray Module, Rechargeable Battery
- **Software:** Arduino IDE, Embedded C/C++, Android Mobile Application
- **Communication:** Bluetooth Low Energy (BLE)
- **Platform:** Internet of Things (IoT)
- **Version Control:** Git & GitHub
